## 🚀 How to use

データはCloudflare Pagesを用いて配信する予定です。
プログラムから用いる際はそちらを使用するようにしてください。

## ⛏️   Development

本リポジトリのプログラムの作成には[aider](https://aider.chat/)を使用しています。

aiderに関する情報は`.aiderrules`に記述してあります。
`.aider.config.yml`で以下のように記述することでこれらの情報を有効に活用できます。

```yaml
read:
  - .aiderrules
```
